# 📩 SMS Spam Message Classification Using LSTM Neural Network

## 📌 Project Overview
This project focuses on building a deep learning model to classify SMS messages as Spam or Ham (Not Spam) using a Long Short-Term Memory (LSTM) Neural Network.

The model processes text messages using Natural Language Processing (NLP) techniques such as tokenization, padding, and sequence modeling to learn patterns in spam messages.

This project demonstrates the application of Deep Learning and NLP for real-world text classification problems.



## 🎯 Objectives

- Classify SMS messages into Spam or Ham
- Apply Natural Language Processing (NLP)
- Build an LSTM-based deep learning model
- Evaluate model performance
- Demonstrate sequence modeling

## 📂 Dataset

The dataset contains labeled SMS messages categorized as:

- Spam → Unwanted messages
- Ham → Normal messages

Preprocessing steps:

- Text cleaning
- Lowercasing
- Removing stopwords
- Tokenization
- Padding sequences



## ⚙️ Model Architecture

The deep learning model includes:

- Embedding Layer
- LSTM Layer
- Dense Layer
- Output Layer (Sigmoid Activation)



## 🔄 Workflow

1. Load Dataset  
2. Clean Text Data  
3. Tokenize Messages  
4. Convert Text to Sequences  
5. Pad Sequences  
6. Build LSTM Model  
7. Train Model  
8. Evaluate Model  
9. Predict Spam Messages
